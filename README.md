# SoundCloud Redux

A basic SoundCloud API client built with React, Redux, and Redux Saga. Try the [live demo](https://soundcloud-redux.herokuapp.com).


## Stack
- React
- React Redux
- React Router
- Redux Saga
- RxJS
- Immutable
- Ava
- Babel
- Circle CI
- Express
- Heroku
- Karma
- Superagent
- Webpack
